# House Price Prediction Using Machine Learning

This project predicts house prices using Machine Learning techniques and real estate data analysis. The model is trained using Linear Regression to estimate house prices based on features like bedrooms, bathrooms, living area, floors, and lot size.

## Project Overview
The main goal of this project is to analyze housing data and build a predictive model that can accurately estimate house prices.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Linear Regression Model
- House Price Prediction
- Data Visualization

## Dataset
The dataset contains housing information such as:
- Bedrooms
- Bathrooms
- Square Foot Living Area
- Floors
- Lot Size
- House Price

## Machine Learning Algorithm
- Linear Regression

## Model Evaluation
The model performance is evaluated using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R2 Score
